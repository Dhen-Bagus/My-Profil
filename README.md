# 🚀 Hadi Bagus Indrawan

Website portfolio pribadi yang saya bangun sebagai media untuk menampilkan perjalanan belajar, pengalaman kerja, dan proses pengembangan kemampuan saya di bidang teknologi dan programming.

Portfolio ini dibuat dengan konsep modern responsive design menggunakan HTML, CSS, dan JavaScript, serta menjadi tempat saya mendokumentasikan perkembangan skill secara bertahap melalui project nyata dan pengalaman langsung di lapangan.

---

# 👨‍💻 Tentang Saya

Saya adalah mahasiswa Teknik Informatika Universitas Budi Luhur sekaligus seseorang yang memulai perjalanan di dunia programming benar-benar dari dasar.

Sebagai lulusan SMK non-programming dan mahasiswa kelas karyawan yang harus menjalani dunia kerja sambil kuliah, proses belajar teknologi tidak selalu mudah untuk dijalani. Keterbatasan waktu belajar, aktivitas kerja lapangan, serta sistem pembelajaran kelas karyawan yang terbagi antara offline dan online membuat saya harus belajar lebih mandiri di luar perkuliahan.

Sebagian besar proses pengembangan kemampuan saya dibangun melalui:
- Belajar mandiri
- Eksplorasi AI tools
- Google & YouTube
- Dokumentasi teknologi
- Trial & error
- Pengalaman project nyata

Melalui proses tersebut saya mulai memahami bahwa perkembangan kemampuan tidak hanya berasal dari teori di kelas, tetapi juga dari rasa ingin tahu, konsistensi belajar, keberanian mencoba, dan pengalaman nyata di lapangan.

---

# 📚 Perjalanan Belajar

Portfolio ini bukan hanya tentang hasil akhir project, tetapi juga tentang perjalanan saya dalam berkembang sedikit demi sedikit di dunia teknologi modern.

Saya menyadari kemampuan saya saat ini masih terus berkembang, namun setiap project yang saya kerjakan menjadi proses pembelajaran baru dalam memahami:
- Frontend Development
- Mobile Development
- UI/UX
- API Integration
- Problem Solving
- Workflow System
- Debugging
- Team Collaboration

Dalam proses pengembangannya, saya juga banyak memanfaatkan AI tools untuk membantu proses belajar, memahami alur coding, debugging, eksplorasi ide, dan mempercepat workflow development modern.

Bagi saya, AI bukan pengganti proses belajar, tetapi menjadi tools pendukung untuk membantu saya berkembang lebih cepat dan memahami teknologi dengan lebih luas.
