🚀 Demo / Buka Project

Absensi berbasis Firebase Yayasan Manunggal Bima Suci

Sistem absensi digital berbasis web yang dikembangkan untuk kebutuhan operasional Yayasan Manunggal Bima Suci dengan konsep role-based access management.

Sistem memiliki beberapa level hak akses seperti Direktur, Pengurus, dan Karyawan. Direktur memiliki akses penuh untuk mengelola data karyawan dan monitoring absensi. Pengurus dapat melakukan absensi sekaligus melihat data kehadiran karyawan. Sedangkan karyawan hanya dapat melakukan absensi masuk, keluar, dan melihat riwayat absensinya sendiri.

Project ini dikembangkan menggunakan HTML, CSS, JavaScript, Firebase, serta dashboard monitoring realtime dengan tampilan responsive untuk desktop maupun mobile.

Meskipun project ini belum digunakan secara penuh dalam operasional yayasan karena proses implementasi terlambat setelah kegiatan proyek yayasan selesai, project ini menjadi pengalaman penting dalam mempelajari sistem absensi digital, role management, realtime database, dan alur kerja monitoring berbasis web.

WEB & Aplikasi Monitoring dan Laporan kerja Kunjunggan PT Beauty World Indonesia

Web dan aplikasi mobile monitoring kunjungan teknisi ini dikembangkan berdasarkan pengalaman kerja langsung di lapangan saat bekerja sebagai teknisi di PT Beauty World Indonesia.

Sebelum sistem ini dibuat, proses absensi kunjungan, dokumentasi pekerjaan, dan laporan teknisi masih dilakukan secara manual melalui grup WhatsApp sehingga data sering tercampur dan sulit dimonitor.

Dari kondisi tersebut, saya mulai menganalisa kebutuhan sistem monitoring yang lebih terstruktur dan mengajukan ide digitalisasi kunjungan teknisi kepada Manager IT perusahaan hingga akhirnya disetujui menjadi project nyata berbasis React Native dan Expo.

Perusahaan juga mendukung pengembangan sistem ini dengan penggunaan Google Maps API berbayar untuk validasi lokasi, tracking realtime, dan integrasi titik lokasi customer secara lebih akurat.

Sistem mendukung workflow operasional nyata mulai dari Team Leader membuat tugas kunjungan, teknisi melakukan absensi berdasarkan validasi lokasi realtime, selfie otomatis dengan metadata lokasi dan waktu, hingga pelaporan kerja digital yang terintegrasi dengan backend API dan ERP perusahaan.

Salah satu bagian paling kompleks dalam project ini adalah sistem statistik dan monitoring performa teknisi untuk kebutuhan operasional perusahaan secara realtime.

Dalam proses pengembangannya, saya juga banyak memanfaatkan AI tools untuk membantu proses belajar, debugging, eksplorasi ide, dan mempercepat workflow development. Meskipun kemampuan saya saat ini masih terus berkembang, pengalaman project ini membuat saya semakin termotivasi untuk terus belajar agar kedepannya mampu membangun dan memahami sistem secara lebih mandiri dan lebih baik lagi.
